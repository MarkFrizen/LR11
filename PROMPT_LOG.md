## Задание B7: Создать multi-stage сборку для Python-приложения с Rust-расширением.
### Промпт 1
**Промпт:** "Напиши multi-stage Dockerfile для Python-приложения, которое использует Rust-расширение (pyo3 + maturin). Первый этап: компиляция Rust-крейта. Второй этап: сборка Python-колеса с помощью maturin. Третий этап: финальный образ на python:3.12-slim, куда копируется только колесо и устанавливается."
**Результат:** Итоговый multi-stage Dockerfile:
Этап 1: rust-builder (rust:1.88-slim)
- COPY Cargo.toml + src/
- RUN cargo build --release
- → Компилирует Rust-крейт (cdylib для Python)
- ↓ COPY target/
Этап 2: wheel-builder (python:3.12-slim + gcc)
- pip install maturin
- COPY Cargo.toml + src/ + pyproject.toml
- COPY --from=rust-builder target/ ← кэш крейтов
- RUN maturin build --release --out dist
- → Собирает .whl колесо с Rust-расширением
- ↓ COPY dist/*.whl
Этап 3: runtime (python:3.12-slim)
- pip install --no-cache-dir dist/*.whl
- COPY main.py
- → Только wheel + код, нет gcc/rust/cargo
### Промпт 2
**Промпт:** "Поправь пути: Rust-код лежит в папке rust_extension, Python-код – в python_app. Maturin должен видеть Cargo.toml из папки расширения. Исправь ошибку: "maturin build --manylinux off" не может найти pyproject.toml."
**Результат:** Кэширование по этапам
- rust-builder: target/ с объектами. Инвалидируется при изменении Cargo.toml или исходников Rust.
- wheel-builder: maturin + скачанные зависимости. Инвалидируется при изменении Cargo.toml, pyproject.toml или исходников.
- runtime: Только .whl файл. Инвалидируется при изменении main.py.
### Промпт 3
**Промпт:** "Уменьши размер финального образа: убери из него все build-зависимости (rustc, cargo, gcc), оставь только python:3.12-slim и установленный wheel. Также добавь проверку, что Rust-функция корректно вызывается из Python (например, эндпоинт /rust_hello)."
**Результат:** Финальный образ (runtime) содержит только:
- python:3.12-slim — базовый Python
- Установленный .whl с Rust-расширением
- main.py — код приложения
### Итого
- Количество промптов: 3
- Время: ~35 мин
